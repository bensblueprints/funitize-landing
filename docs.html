Custom Animations

Upload your own images to create truly unique falling animations:

  1. Click "Upload Custom Animation"
  2. Select an image from your computer (PNG recommended, max 500KB)
  3. The image will fall from the top of your store
  4. Customize size, speed, and frequency
Best Practices: Use transparent PNG images for best results. Recommended size: 50x50 to 200x200 pixels.

Clickable Popups (CTAs)

Turn your animations into interactive call-to-actions:

  1. Enable "Clickable Popup" when creating an animation
  2. Set the click destination URL (e.g., /collections/sale)
  3. Add optional text overlay on the animation
  4. Choose popup frequency (every animation or random)

Scheduling

Plan your animations ahead of time for holidays, sales, and special events:

  1. Create your animation
  2. Click "Schedule" instead of "Activate Now"
  3. Set start date and time
  4. Set end date and time (optional)
  5. Save - the animation will activate automatically

Example: Schedule snowfall for December 1st - December 25th, hearts for Valentine's week, and pumpkins for Halloween.

Page Targeting

Show different animations on different pages:

  • All Pages: Animation appears site-wide
  • Homepage Only: Only on your main landing page
  • Product Pages: Only on /products/* pages
  • Collections: Only on collection listing pages
  • Cart Page: Only on the cart/checkout
  • Custom URL: Enter specific URLs

‚ö° Performance & Best Practices

Does Funitize Slow Down My Store?

No. Funitize is optimized for performance:

  • Animations use hardware-accelerated CSS and Canvas
  • Lazy loading prevents animations from affecting page load time
  • Automatic throttling on mobile devices
  • Total script size: <20KB (minified and gzipped)

Recommendations

  • Don't overdo it: 1-2 active animations work best
  • Match your brand: Use animations that fit your store's vibe
  • Seasonal relevance: Rotate animations with seasons/holidays
  • Mobile consideration: Preview on mobile - heavy animations auto-reduce

üîß Troubleshooting

Animation Not Showing

  • Check that the animation is toggled "Active" in your dashboard
  • Verify page targeting - it might be set to show on specific pages only
  • Clear your browser cache (Cmd+Shift+R or Ctrl+Shift+R)
  • Check if any conflicting apps are blocking scripts

Animation Looks Different on Mobile

This is intentional. Funitize automatically reduces particle count and animation intensity on mobile devices to preserve performance and battery life.

Custom Image Not Working

  • Ensure image is under 500KB
  • Use PNG or SVG format (JPG works but no transparency)
  • Verify image is not corrupted by re-uploading
